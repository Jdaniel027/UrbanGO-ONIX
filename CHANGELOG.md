## [0.1.0] - 2026-02-12

### Agregado
- Arquitectura base del proyecto
- Sistema de mapas con Mapbox
- Documentación completa:
  - Arquitectura de componentes
  - Estructura de carpetas
  - Estándares de código
  - Flujo de Git
  - Guía de contribución
  - Gestión de dependencias
- Módulos del mapa:
  - POIs (Puntos de Interés)
  - Routes (Rutas de camiones)
  - Stops (Paradas)
  - Vehicle (Tracking de vehículos)
- Configuración de desarrollo:
  - Expo con React Native
  - TypeScript
  - ESLint
  - Mapbox integrado
- Setup para desarrollo con APK precompilado
- Roadmap del proyecto
- Documentación de API (contratos)
- Guía de trabajo en equipo
- Guía de testing
- Política de seguridad

### Tecnologías

## Tipos de Cambios

- **Agregado** - para nuevas funcionalidades
- **Cambiado** - para cambios en funcionalidades existentes
- **Obsoleto** - para funcionalidades que se eliminarán pronto
- **Eliminado** - para funcionalidades eliminadas
- **Corregido** - para corrección de errores
- **Seguridad** - en caso de vulnerabilidades

## Versionamiento

Este proyecto usa [Semantic Versioning](https://semver.org/lang/es/):
- **MAJOR** (X.0.0) - Cambios incompatibles con versiones anteriores
- **MINOR** (0.X.0) - Nuevas funcionalidades compatibles
- **PATCH** (0.0.X) - Correcciones de bugs compatibles
